Cette documentation explique le fonctionnement et les composants du code fourni.

HTML

Le code HTML structure la page avec un conteneur principal pour des éléments interactifs.

<div class="test1">
    <div>
        <a href="Class/log_off.php">Déconnexion</a>
    </div>
    <div id="update_1__update_1_text01" onclick="general_send(this)" 
        class="update_1__root__id_sha1_user__NEUTRE__id_user__5">
        ADD
    </div>
</div>

<textarea name="" id="update_1_text01">
Lorem ipsum dolor sit amet, consectetur adipisicing elit.
</textarea>

Description :

  • Un bouton de déconnexion pointe vers un fichier PHP.
  • Un div déclenche une fonction JavaScript general_send lors du clic.
  • Une zone de texte textarea contient un exemple de texte.

JavaScript

Le script JavaScript gère les interactions utilisateur et les envois de données.

function afficherValeursFormattees(chaine) {
    const valeurs = chaine.split("__");
    return valeurs;
}

function general_send(_this) {
    const texteFormatte = afficherValeursFormattees(_this.className);
    const texteFormatte2 = afficherValeursFormattees(_this.id);

    var ok = new Information("function/general.php");
    ok.add("option00", "");

    if (texteFormatte[1] != undefined) {
        ok.add("option0_2", texteFormatte[1]);
    }
    if (texteFormatte[0] != undefined) {
        ok.add("option0_1", texteFormatte[0]);
        if (texteFormatte2[0] == texteFormatte[0]) {
            var value = document.getElementById(texteFormatte2[1]).value;
            var new_val = value.replace(/'/g, "'").replace(/"/g, """);
            ok.add("option1_2", new_val);
        }
    }
    console.log(ok.info());
    ok.push();
}

Description :

  • La fonction afficherValeursFormattees divise une chaîne par "__".
  • La fonction general_send récupère des informations des classes et ID HTML.
  • Les données sont formatées, loggées dans la console, puis envoyées via une instance Information.

CSS

Le style CSS applique une mise en forme et des transitions pour les éléments HTML.

.test1 {
    display: flex;
    justify-content: space-around;
    background-color: black;
    transition: 1s all;
}

.test1 div {
    padding: 15px;
    color: green;
    border: 1px solid rgba(255, 255, 255, 0.6);
}

.test1 div:hover {
    background-color: white;
    color: black;
    cursor: pointer;
}

Description :

  • Disposition flexible des éléments avec flexbox.
  • Effet de survol : fond blanc, texte noir, et transition fluide.
  • Style distinct pour les liens avec texte rouge.
