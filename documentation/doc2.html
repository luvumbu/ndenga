Classe PHP : Creat_form

Description

La classe Creat_form permet de générer dynamiquement des formulaires HTML et leurs éléments enfants en attribuant des propriétés et des événements personnalisés.

Méthodes

1. __construct($id, $tag, $style)

Initialise un nouvel objet Creat_form.

  • $id : ID unique du formulaire (type : string).
  • $tag : Balise HTML principale (par défaut : "form").
  • $style : Classe ou styles CSS optionnels.
$form = new Creat_form("formID", "form", "my-style");

2. stylesheet($path)

Ajoute une feuille de style externe au formulaire.

  • $path : Chemin vers le fichier CSS (type : string).
$form->stylesheet("style.css");

3. add_child($id, $tag)

Ajoute un élément enfant au formulaire.

  • $id : ID unique de l'élément enfant (type : string).
  • $tag : Type d'élément HTML (par ex. : "input", "button").
$form->add_child("inputName", "input");

4. child_setAtribut($child_id, $attribute, $value)

Définit un attribut pour un élément enfant spécifique.

  • $child_id : ID de l'enfant (type : string).
  • $attribute : Nom de l'attribut HTML (par ex. : "placeholder").
  • $value : Valeur de l'attribut (type : string).
$form->child_setAtribut("inputName", "placeholder", "Entrez votre nom");

Exemple d'Utilisation


$form = new Creat_form("monForm", "form", "");
$form->stylesheet("styles.css");
$form->add_child("nom", "input");
$form->child_setAtribut("nom", "placeholder", "Entrez votre nom");
$form->add_child("envoyer", "button");
$form->child_setAtribut("envoyer", "innerHTML", "Envoyer");
